# Utils
> 특정 기능이 필요하지 않은  
> 자주 사용하는 메서드 정의
> - ## BeanUtil
>> Bean 객체를 조회하는 기능 정의  
>> 최대한 사용하지 않는 것을 권장
> - ## CryptoUtil
>> Encoding, Decoding, Encrypt, Decrypt 기능 정의
> - ## DateUtil
>> 날짜 관련 기능 정의
> - ## MapperUtil
>> Model Mapper 기능 정의
> - ## StringUtil
>> 문자 관련 기능 정의
> - ## XssUtil
>> XSS 기능 정의
